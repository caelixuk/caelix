# Code of Conduct

CAELIX is an open research project. Participation is welcome, but it must remain constructive.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any other community channels used for CAELIX.

## Expected behaviour

- Be respectful and professional.
- Critique ideas and implementations, not people.
- Keep discussion evidence-led (reproductions, logs, diffs, references).
- Assume good faith, but do not tolerate harassment or intimidation.

## Unacceptable behaviour

- Harassment, discrimination, or personal attacks.
- Trolling, derailing, or repeatedly posting in bad faith.
- Doxxing or sharing private information.
- Misrepresenting results, provenance, or authorship.

## Enforcement

The Maintainer may remove, edit, or reject contributions (including issues, comments, and pull requests) that violate this Code of Conduct.
